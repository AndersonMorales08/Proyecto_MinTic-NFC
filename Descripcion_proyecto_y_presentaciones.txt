Descripción Proyecto
    Analizar, diseñar y construir una aplicación de software que permita controlar las 
    ventas físicas y virtuales de una empresa de manufactura y hacerles el 
    correspondiente seguimiento. 
    Esta aplicación se realizará en un grupo de 5 personas y se usarán metodologías ágiles,
    en específico, SCRUM. Cada integrante deberá tener un rol específico dentro del grupo:
        1. Desarrollador
        2. Analista
        3. Product Owner
        4. Administrador de bases de datos
        5. Scrum Master
    Los roles se irán rotando al paso de las semanas como se específico dentro del grupo.

    Para la realización del proyecto se utilizarán los lenguajes de programación: JavaScript
    y TypeScript, con el siguiente stack tecnológico:
        - Stack MERN:
            * Mongo DB.
            * Express Js.
            * React Js.
            * Node Js.
        - HTML 5.
        - CSS3.
        - Plataforma de desarrolo para Backend (Node Js).
        - OAuth2.
        - Heroku.
        - Git/GitHub.
